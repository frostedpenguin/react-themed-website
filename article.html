Εισαγωγή στο React

Tι είναι το React

Το React είναι μία βιβλιοθήκη της javaScript οπου αναπτύχθηκε απο το faceboook.Χρησιμοποιείται για την ανάπτυξη διεπαφών χρήστη για ιστοσελίδες.Το κυρίως προτέρημα του είναι η δυνατότητα που προσφέρει στους προγραμματιστές να διαιρούν την ιστοσελίδα σε ανεξάρτητα μέρη (components) οπου αλληλεπιδρούν μεταξύ τους και μαζι συνθέτουν το τελικό αποτέλεσμα

Το React εχει αλλάξει τον τροπο με τον οποίο σκεφτόμαστε τις εφαρμογές ιστου και μας έχει δώσει την δυνατότητα να αναπτύσσουμε και να συντηρούμε ιστοσελίδες μεγάλης κλίμακας.

Πως δουλεύει;

Ενα αναπόσπαστο κομμάτι της λειτουργικότητάς του react ειναι οτι δεν διαχειρίζεται το DOM του φυλλομετρητή αλλα δημιουργεί ενα εικονικό DOM (virtual DOM) στην μνημη. Όποιες αλλαγές γίνουν στο εικονικό DOM εφαρμόζονται στο DOM του φυλλομετρητή του browser υστερά.

Components

Τα components είναι τα βασικα στοιχεία μιας εφαρμογής React. Ειναι τα ανεξάρτητα μέρη οπου αναφέρθηκα παραπάνω. Ενα παράδειγμα αποτελεί η παρακάτω φωτογραφία.
Κάθε ενα πλαίσιο αντιπροσωπεύει ενα component. Το καθένα απο αυτα είναι υπεύθυνο για την εμφανισή του και τις αλληλεπιδράσεις του. Προγραμματιστικά ενα component συνθέτεται απο στοιχεία html και το styling τους ,καθώς και την λογικη javaScript

JSX

Το React ομως οπως αναφερθήκαμε είναι μια βιβλιοθήκη javaScript οπου απο μόνη της σαν γλώσσα δεν χρησιμοποιείται για την δημιουργία στοιχείων. Για αυτο το React δημιούργησε την JSX. Το κοινό hello world εκφράζεται με τον εξής τρόπο: 

           const element = <h1>Hello, world!</h1>;

Όπως μπορείτε να καταλάβετε είναι σαν ένα υβρίδιο html με javaScript. Στην ουσία είναι ένα extension της γλώσσας όπου μας επιτρέπει να αναθέτουμε τιμές html σε μεταβλητές

Δομή Component

Ενα παράδειγμα δομής ενος component είναι το εξής :

       class Welcome extends React.Component {
        render() {
          return <h1>Hello, {this.props.name}</h1>;
        }
      }

Καθε component ειναι στην ουσία μια κλαση ES6 οπού μας επιστρέφει ενα στοιχείο JSX. Η συνάρτηση render καλείτε αρχικά στην αρχικοποίηση του component και ετσι "συλλέγεται" απο το virtual dom και εμφανίζεται στην ιστοσελίδα. Η λέξη props είναι δεσμευμένη, ειναι ένα αντικείμενο που χρησιμοποιείται για να περάσει δεδομένα μεσα στο component .Βρίσκεται μεσα σε αγκύλες γιατί ειναι δυναμικά αρχικοποιημένο κάθε αν αλλάξουν τα δεδομένα εισόδου αλλάζει και το στοιχείο. Αλλα επειδή η συνάρτηση render δεν ξανακαλείτε η αλλαγή γίνεται μόνο σε τοπικό επίπεδο και δεν αλλάζει κατι στην ιστοσελίδα.

State

Αμα θέλαμε για παράδειγμα να αλλάζαμε δυναμικά το όνομα στο παραπάνω παράδειγμα, έπρεπε να ταν μέρος του state του component. Το state ειναι μια συλλογή μεταβλητών της κλάσης που αρχικοποιείται στον constructor και εκφράζει ακριβώς την κατάσταση του όλου στοιχείου. κάθε φορα που αλλάζει η κατάσταση η συνάρτηση render καλείται και η αλλαγή αποτυπώνεται στο DOM. Ενα παράδειγμα εφαρμογής του state είναι το εξής: 

      
        class Clock extends React.Component { constructor(props) { super(props);
        this.state = {date: new Date()}; } render() { return (
        <div>
          <h1>Hello, world!</h1>
          <h2>It is {this.state.date.toLocaleTimeString()}.</h2>
        </div>
        ); } }

Το component αυτο εκφράζει ενα ρολόι αρχικοποιεί το state με την τωρινη ώρα και την εμφανίζει απο κάτω. Προσοχή ομως καποιος μπορεί να πιστεύει οτι το ρολοί αυτο αλλαζει την ωρα σε πραγματικό χρόνο αλλα εμεις απλα δε του αλλάζουμε την αξία καθε δευτερόλεπτο οποτε απλά μας δείχνει την ώρα αρχικοποίησης. Αμα θελεται να δείτε πως θα πετύχουμε αυτο το αποτέλεσμα πατήστε εδώ.
